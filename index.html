Introduction

This course is about learning how to communicate with the computer so it can help you achieve your goals. it does't matter if you are an artist or a scientist computer programming can help you achieve your goals.

What we are going to learn at this course

Stage 1 - Basics of computer programming -

  • Learn how to read binary (Computer language) .
  • Learn What is Computer programming ? .
  • Learn The basics of computer programming ?.
  • Head start to the Java programming language.
  • Small project to demonstrate what you just learn.

Stage 2 -Object oriented programming

  • Learn about cloud computing .
  • Dive into Object oriented programming.
  • Make another project to demonstrate stage one and stage 2.

Stage 3 - Swing basics and JavaFX

  • learn how to make a user interface and awesome animation using the javaFX technology and Swing API.
  • yet another project.

Stage 4 - Object oriented analysis and design

  • Learn how to communicate people to make big ideas come to live .
  • Making your idea into a computer program.

Stage 5 - Design pattern

  • Learn a common issues in software development and how to solve this problems.
